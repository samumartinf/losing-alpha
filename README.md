# Losing Alpha
You are still losing money, but now you know why. Allowing users to understand their profit, loss and risk. 

## Development

```bash
pnpm dev
```

## Database

```bash
pnpm db:start
```

### Migrations

```bash
pnpm db:migrate
```

### Studio

```bash
pnpm db:studio
```
